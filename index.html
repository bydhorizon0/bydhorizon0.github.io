멀티프로세서 기반 PDF 변환 프로그램

각 사이트에 첨부된 hwp, xlsx(엑셀), docx(워드), txt, pdf, jpg 등 의 파일들을 pdf 파일로 변환하는 프로젝트입니다.

COM 객체를 쉽게 사용할 수 있는 파이썬을 사용해 prefix로 설정한 폴더를 대상으로 변환 설정한 확장자를 pdf 파일로 변환합니다.

COM 객체를 다루기 위해 pywin32라는 오픈 소스를 사용했으며 파일 변환은 주로 I/O bound 가 대부분이지만 엑셀의 빠른 셀 처리를 위해 멀티프로세싱을 사용했습니다.

정책 차트 사이트

수집된 공고들의 현황을 보여주는 사이트입니다. 사용한 기술로는 Spring boot, React, Highcharts library 입니다.

처음 React를 사용해보는 프로젝트로 React의 중요한 개념인 Hooks를 학습하는데 큰 도움이 됐습니다.

Docker compose를 사용했으며 API 서버, Frontend, DB를 각각 컨테이너로 만들어 적용했습니다.

비동기 공고 수집기

사용한 기술로는 Spring boot, Selenium 입니다. 자동화 탐지 회피, robots, 개발자모드 접근 제한 등 웹 상의 고려사항을 최대한 회피하여 수집했습니다.

자동화 탐지 회피를 위해 스크래핑 중간에 랜덤값을 주어 대기합니다. 동기식 스크래핑은 네트워크 I/O에 상당한 시간이 소모되어 수집 시간이 상당히 오래걸립니다.

동기식의 단점을 보완하고자 ExecutorService로 멀티쓰레드를 사용해 다수의 사이트를 비동기로 수집했습니다.

충북 & 충남 지역 데이터 시스템

사용한 기술로는 Spring boot와 EChart.js 를 사용해서 충북과 충남 지역의 데이터를 시각화하는 시스템입니다.
